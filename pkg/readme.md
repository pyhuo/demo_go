## 2.变量生命周期
* golang 可以返回局部变量的地址，gc会通过引用计数释放.
* 包一级声明的变量: 它们的生命周期和整个程序的运行周期是一致的
* 局部变量的生命周期则是动态的: 每次从创建一个新变量的声明语句开始，直到该变量不再被引用为止，然后变量的存储空间可能被回收
* 函数的参数变量和返回值变量都是局部变量: 它们在函数每次被调用的时候创建
